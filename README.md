# Java Vocabulary 📚
## 1. Структура кода и комментарии (Code Structure & Comments)

| Термин | Транскрипция | Перевод / Значение в Java |
| :--- | :--- | :--- |
| **Class** | `[klɑːs]` | **Класс** — шаблон/чертёж для написания кода. |
| **Method** | `[ˈmeθəd]` | **Метод** — функция, блок кода для выполнения действия. |
| **Main** | `[meɪn]` | **Главный** — точка входа, откуда запускается программа. |
| **Package** | `[ˈpækɪdʒ]` | **Пакет** — папка для группировки классов. |
| **Syntax** | `[ˈsɪntæks]` | **Синтаксис** — правила написания кода (грамматика языка). |
| **Comment** | `[ˈkɒment]` | **Комментарий** — текст в коде, который игнорирует компилятор. |
| **Single-line** | `[ˈsɪŋɡl laɪn]` | **Однострочный** — комментарий через `//`. |
| **Multi-line** | `[ˈmʌlti laɪn]` | **Многострочный** — комментарий через `/* ... */`. |
| **Javadoc** | `[ˈdʒɑːvəˌdɒk]` | **Джавадок** — документирующий комментарий `/** ... */`. |
| **Source Code** | `[sɔːs kəʊd]` | **Исходный код** — текст программы, написанный программистом. |
| **Compiler** | `[kəmˈpaɪlə]` | **Компилятор** — программа, переводящая код в байт-код. |

## 2. Переменные и типы данных (Variables & Data Types)

| Термин | Транскрипция | Перевод / Значение в Java |
| :--- | :--- | :--- |
| **Variable** | `[ˈveəriəbl]` | **Переменная** — «коробка» в памяти для хранения данных. |
| **Declare** | `[dɪˈkleə]` | **Объявить переменную** — создать её в коде (выделить под неё «коробку»), указав тип и имя. |
| **Initialize** | `[ɪˈnɪʃəlaɪz]` | **Инициализировать** — записать в переменную её самое первое значение. |
| **Type inference** | `[taɪp ˈɪnfərəns]` | **Выведение типа** — автоматическое определение типа компилятором через ключевое слово `var`. |
| **Bit** | `[bɪt]` | **Бит** — минимальная единица памяти (`0` или `1`). |
| **Byte** | `[baɪt]` | **Байт** — ячейка памяти из 8 бит. |
| **Primitive types** | `[ˈprɪmətɪv taɪps]` | **Примитивные типы** — базовые встроенные типы данных (`byte`, `short`, `int`, `long`, `float`, `double`, `char`, `boolean`). |
| **Integer** | `[ˈɪntɪdʒə]` | **Целое число** — группа целочисленных типов (в частности тип `int`). |
| **Floating-point** | `[ˈfləʊtɪŋ pɔɪnt]` | **Плавающая точка** — дробные числа (`float`, `double`). |
| **Double precision** | `[ˈdʌbl prɪˈsɪʒn]` | **Двойная точность** — стандартный дробный тип `double` (64 бита). |
| **Character (Char)** | `[ˈkærəktə]` | **Символ** — одиночный знак в одинарных кавычках `' '`. |
| **String** | `[strɪŋ]` | **Строка** — текст (ссылочный тип данных) в двойных кавычках `" "`. |
| **Boolean** | `[ˈbuːliən]` | **Логический / Булев тип** — тип данных, принимающий только `true` или `false`. |

## 3. Операции и приведение типов (Operations & Casting)

| Термин | Транскрипция | Перевод / Значение в Java |
| :--- | :--- | :--- |
| **Concatenation** | `[kɒnˌkætɪˈneɪʃn]` | **Конкатенация** — склеивание строк или текста через оператор `+`. |
| **Parse** | `[pɑːs]` / `[pɑːrs]` | **Разбирать / анализировать** — конвертация (парсинг) текста в число (например, `Integer.parseInt`). |
| **Radix** | `[ˈreɪdɪks]` | **Основание** системы счисления (например, `2` для двоичной, `10` для десятичной). |
| **Binary** | `[ˈbaɪnəri]` | **Двоичный** — относящийся к двоичной системе счисления (`0` и `1`). |
| **Type Casting** | `[taɪp ˈkɑːstɪŋ]` | **Приведение типов** — процесс преобразования значения переменной одного типа в другой тип. |
| **Widening** | `[ˈwaɪdnɪŋ]` | **Расширение** — автоматическое (безопасное) приведение от меньшего типа к большему. |
| **Narrowing** | `[ˈnærəʊɪŋ]` | **Сужение** — ручное (опасное) приведение `(тип)` от большего типа к меньшему. |
| **Truncation** | `[trʌŋˈkeɪʃn]` | **Усечение** — потеря дробной части (отбрасывание хвоста) при кастинге дробного числа в целое. |
| **Overflow** | `[ˌəʊvəˈfləʊ]` | **Переполнение** — потеря данных, когда число превышает лимит типа и срезается по границе байтов. |

## 4. Финализация и Ввод-Вывод (Finalization & I/O)

| Термин | Транскрипция | Перевод / Значение в Java |
| :--- | :--- | :--- |
| **Final** | `[ˈfaɪnl]` | Ключевое слово, которое превращает переменную в неизменяемую константу. |
| **Constant** | `[ˈkɒnstənt]` | **Константа** — величина, значение которой нельзя изменить в ходе выполнения программы. |
| **Immutable** | `[ɪˈmjuːtəbl]` | **Неизменяемый** — объект или переменная, чьё состояние защищено от модификации после создания. |
| **Mutable** | `[ˈmjuːtəbl]` | **Изменяемый** — объект, состояние или значение которого можно свободно менять. |
| **UPPER_CASE** | `[ˈʌpə keɪs]` | **Верхний регистр** — общепринятый стиль написания имён констант в Java (например, `MAX_VALUE`). |
| **I/O (Input/Output)** | `[ˈɪnpʊt ˈaʊtpʊt]` | **Ввод/Вывод** — процесс обмена данными между программой и внешним миром (консолью, файлами). |
| **Stream** | `[striːm]` | **Поток** — метафора в Java для передачи данных (как капли воды по трубе) от источника к получателю. |
| **Buffer** | `[ˈbʌfə]` | **Буфер** — временная область памяти, куда собираются данные перед отправкой, чтобы ускорить работу `I/O`. |

## 5. Операторы и инкременты (Operators & Increments)

| Термин | Транскрипция | Перевод / Значение в Java |
| :--- | :--- | :--- |
| **Arithmetic Operators** | `[ˌærɪθˈmetɪk ˈɒpəreɪtəz]` | **Арифметические операторы** — знаки математических действий (`+`, `-`, `*`, `/`, `%`). |
| **Modulo (Remainder)** | `[ˈmɒdjʊləʊ]` | **Остаток от деления** (`%`) — остаток, получаемый при делении одного целого числа на другое. |
| **Increment** | `[ˈɪŋkrɪməнт]` | **Инкрементирование** — увеличение значения числовой переменной на 1 (`++`). |
| **Decrement** | `[ˈdekrɪməнт]` | **Декрементирование** — уменьшение значения числовой переменной на 1 (`--`). |
| **Prefix Form** | `[ˈpriːfɪks fɔːm]` | **Префиксная форма** (`++x`) — сначала изменяет переменную в памяти, а затем отдает значение в выражение. |
| **Postfix Form** | `[ˈpəʊstfɪks fɔːm]` | **Постфиксная форма** (`x++`) — сначала отдает текущее значение в выражение, а переменную изменяет "на будущее". |
| **Assignment Operator** | `[əˈsaɪnmənt ˈɒpəreɪtə]` | **Оператор присваивания** (`=`) — базовый оператор для записи значения в переменную. |
| **Compound Assignment** | `[kəmˈpaʊnd əˈsaɪnmənt]` | **Составное присваивание** (`+=`, `-=`, `*=`) — совмещение арифметической операции и присваивания. |
| **Comparison Operators** | `[kəmˈpærɪsn ˈɒpəreɪtəz]` | **Операторы сравнения** (`==`, `!=`, `>`, `<`, `>=`, `<=`) — сравнивают значения и возвращают `true` или `false`. |
