# Java Vocabulary 📚

## 1. Структура кода и комментарии (Code Structure & Comments)

| Термин | Транскрипция | Перевод / Значение в Java |
| :--- | :--- | :--- |
| **Class** | `[klɑːs]` | **Класс** — шаблон/чертёж для написания кода. |
| **Method** | `[ˈmeθəd]` | **Метод** — функция, блок кода для выполнения действия. |
| **Main** | `[meɪn]` | **Главный** — точка входа, откуда запускается программа. |
| **Package** | `[ˈpækɪdʒ]` | **Пакет** — папка для группировки классов. |
| **Syntax** | `[ˈsɪntæks]` | **Синтаксис** — правила написания кода (грамматика языка). |
| **Comment** | `[ˈkɒment]` | **Комментарий** — текст в коде, который игнорирует компилятор. |
| **Single-line** | `[ˈsɪŋɡl laɪn]` | **Однострочный** (комментарий через `//`). |
| **Multi-line** | `[ˈmʌlti laɪn]` | **Многострочный** (комментарий через `/* ... */`). |

## 2. Переменные и типы данных (Variables & Data Types)

| Термин | Транскрипция | Перевод / Значение в Java |
| :--- | :--- | :--- |
| **Variable** | `[ˈveəriəbl]` | **Переменная** — «коробка» в памяти для хранения данных. |
| **Type inference** | `[taɪp ˈɪnfərəns]` | **Выведение типа** — автоматическое определение типа через `var`. |
| **Bit** | `[bɪt]` | **Бит** — минимальная единица памяти (`0` или `1`). |
| **Byte** | `[baɪt]` | **Байт** — ячейка памяти из 8 бит. |
| **Primitive types** | `[ˈprɪmətɪv taɪps]` | **Примитивные типы** — базовые встроенные типы данных. |
| **Integer** | `[ˈɪntɪdʒə]` | **Целое число** (тип `int`). |
| **Floating-point** | `[ˈfləʊtɪŋ pɔɪnt]` | **Плавающая точка** — дробные числа (`float`, `double`). |
| **Double precision** | `[ˈdʌbl prɪˈsɪʒn]` | **Двойная точность** — стандартный дробный тип `double`. |
| **Character (Char)** | `[ˈkærəktə]` | **Символ** — одиночный знак в одинарных кавычках `' '`. |
| **String** | `[strɪŋ]` | **Строка** — текст в двойных кавычках `" "`. |
| **Boolean** | `[ˈbuːliən]` | **Булев тип** — логический тип данных (`true` или `false`). |

## 3. Операции и приведение типов (Operations & Casting)

| Термин | Транскрипция | Перевод / Значение в Java |
| :--- | :--- | :--- |
| **Concatenation** | `[kɒnˌkætɪˈneɪʃn]` | **Конкатенация** — склеивание строк или текста через `+`. |
| **Parse** | `[pɑːz]` | **Разбирать/анализировать** — конвертация текста в число. |
| **Radix** | `[ˈreɪdɪks]` | **Основание** системы счисления (например, двоичной). |
| **Binary** | `[ˈbaɪnəri]` | **Двоичный** (двоичный код, двоичная система). |
| **Type Casting** | `[taɪp ˈkɑːstɪŋ]` | **Приведение типов** — превращение одного типа в другой. |
| **Widening** | `[ˈwaɪdnɪŋ]` | **Расширение** — автоматический безопасный кастинг в больший тип. |
| **Narrowing** | `[ˈnærəʊŋ]` | **Сужение** — ручной опасный кастинг `(тип)` в меньший размер. |
| **Truncation** | `[trʌŋˈkeɪʃn]` | **Усечение** — потеря дробной части при кастинге в целое число. |
